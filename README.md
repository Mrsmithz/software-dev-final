# How to run

## for docker compose v2
```
docker compose up
```

## for older docker compose
```
docker-compose up
```

# How to test

## for backend
```
yarn test
```

## for frontend
```
yarn test
```
# How to deploy
## Jenkins Pipeline
```
use Jenkinsfile for Pipeline scripts
```
